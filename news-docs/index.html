From 0 to Progressive Web App in Minutes

Offline support, lazy loading and Service Worker caching out-of-the-box. Designed to be lightweight and flexible, based on web standards. Get started right away using just HTML, CSS and Javascript.

Built with Polymer and Web Components

Polymer News is built out of Web Components, created using the Polymer library. This allows them to be lightweight, modular, re-usable, and interoperable with other web technologies and frameworks.

Supports AMP documents as articles

The app template supports using a simple JSON feed for your article content. If you publish AMP docs, you can use those directly inside of the News shell instead.

Browser Compatibility

Polymer News is based on the Polymer library and App Toolbox, which work in all major browsers (Chrome, Firefox, IE, Edge, Safari, and Opera).
